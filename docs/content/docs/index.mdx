OpenTela is a decentralized distributed computing platform that orchestrates computing resources across a peer-to-peer network. It uses libp2p networking, CRDT-based state management, and gossip protocols.

## Primary Use Case

Distributed GPU node orchestration for LLM serving, powering projects like the SwissAI Initiative.
